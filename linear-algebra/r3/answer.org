* Problem 1
** a
by definiton, we need to prove that $W$ contains $\begin{pmatrix}0\\0\\0\end{pmatrix}$, closed for addition & scalar multiplication
since $0 - 2 \cdot 0 + 3 \cdot 0 = 0$, $W$ does contain $\begin{pmatrix}0\\0\\0\end{pmatrix}$
known $x_1 - 2x_2 + 3x_3 = 0, x_1' - 2x_2' + 3x_3' = 0$, it is obvious that $(x_1 + x_1') - 2(x_2 + x_2') + 3(x_3 + x_3') = 0$
which means $\begin{pmatrix}x_1 + x_1'\\x_2 + x_2'\\x_3 + x_3'\end{pmatrix} \in W$
for scalar multiplication, this is the same with addition
** b
suppose $\vec{u} = \begin{pmatrix}x_1\\x_2\\x_3\end{pmatrix}, \vec{v} = \begin{pmatrix}x_1'\\x_2'\\x_3'\end{pmatrix}$
then $3\vec{u} - 2\vec{v} = \begin{pmatrix}3x_1-2x_1'\\3x_2-2x_2'\\3x_3-2x_3'\end{pmatrix}$
as $\begin{aligned}
3x_1-2x_1' - 2(3x_2-2x_2') + 3(3x_3-2x_3')
&= 3(x_1 - 2x_2 + 3x_3) - 2(x_1' - 2x_2' + 3x_3')\\
&= 0-0\\
&= 0
\end{aligned}$
