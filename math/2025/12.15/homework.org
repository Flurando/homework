to solve this, apply Laplace transform to the funciton
$$L[y''+3y'+2y]=L[f(t)]$$
which is $L[y''] + L[3y'] + L[2y] = L[f(t)]$
$$s^2Y(s) - sy(0) - y'(0) + 3(sY(s) - y(0)) + 2Y(s) = L[f(t)]$$
as we know, $y(0)=y'(0)=0$, there is $s^2Y(s) + 3sY(s) + 2Y(s) = L[f(t)]$
for $L[f(t)]$, by searching online we know it could be represented as $\frac{1}{s} - \frac{e^{-10s}}{s}$
so $Y(s) = \frac{1-e^{-10s}}{s(s^2+3s+2)}$
now we need to inverse the Laplace transform for $Y(s)$
first break it up into $Y(s) = (1-e^{-10s}) \cdot \left(\frac{1}{2s} + \frac{1}{s+1} + \frac{1}{2(s+2)}\right)$
for that with $1$, it is $\frac{1}{2} + e^{-t} + \frac{1}{2}e^{-2t}$
for the other with $e^{-10s}$, it is when $t\geq 10$, yeilding $\frac{1}{2} + e^{10}e^{-t} + \frac{1}{2}e^{-2(t-10)}$
so wrapping up, the solution is
when $t < 10$, $y(t)=\frac{1}{2} + e^{-t} + \frac{1}{2}e^{-2t}$
when $t \geq 10$, $y(t)=(1-e^{10})e^{-t} + \frac{1}{2}e^{-2t} - \frac{1}{2}e^{-2(t-10)}$
